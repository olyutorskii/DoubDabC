Package io.github.olyutorskii.doubdabc Description

DoubDabC is a Java library that supports binary integer value to decimal sequence conversion.

DoubDabC implements Double-Dabble algorithm. Double-Dabble is a radix-conversion algorithm but there is no division(/) nor remainder(%) operation.

There is no String constructor during conversion. So, DoubDabC is a GC-friendly.
